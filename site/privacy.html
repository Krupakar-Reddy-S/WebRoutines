Privacy Policy

Last updated February 2026

The short version

WebRoutines stores all data locally in your browser. Nothing is sent to any server. There are no accounts, no analytics, and no tracking. Your browsing routines stay on your machine.

What data is stored

Routine definitions

Names and ordered URL lists you create. Stored in IndexedDB, a browser-managed local database.

Run history

Start time, duration, steps completed, and completion status for each routine run. Stored in IndexedDB.

Settings

Theme preference, tab loading mode, and feature toggles. Stored in Chrome's local extension storage.

Session state

Active runner state (which routine is running, current step, tab IDs). Stored in session-scoped storage that clears when the browser closes.

What WebRoutines does not do

✕ Does not transmit any data to external servers.
✕ Does not include analytics, telemetry, or tracking of any kind.
✕ Does not require or offer user accounts.
✕ Does not read, collect, or store page content from sites you visit.
✕ Does not inject ads or promotional content.

Permissions and why

WebRoutines requests only the permissions necessary for its features.

Permission Reason
storage Persist routines, run history, and settings locally.
sidePanel Display the main extension interface in Chrome's side panel.
tabGroups Create and manage tab groups for routine runners.
unlimitedStorage Support larger local datasets as run history grows.
host permissions Required for the focus mode controller to appear on web pages. The controller does not read or modify page content — it only renders a navigation overlay.

Data lifecycle

You create it

Routines are created through the extension UI and stored in your browser's IndexedDB.

You control it

Edit, delete, or export routines at any time. Import/export is user-initiated only. There is no background sync.

You remove it

Uninstalling the extension removes all extension-managed data per Chrome's default behavior. Export your routines first if you want to keep them.

This website

This is a static site hosted on GitHub Pages. It does not use cookies, analytics scripts, or any form of user tracking. There are no third-party scripts loaded beyond Google Fonts and the Tailwind CSS CDN.
